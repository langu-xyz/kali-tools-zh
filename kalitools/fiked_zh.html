fiked包装说明

FakeIKEd，或fiked的简称，是假的IKE服务支持刚够的标准和思科扩展到进攻中常见的不安全思科VPN PSK + XAUTH基于IPsec身份验证设置在什么可以被描述为一个半MITM攻击。 Fiked可以为了捕捉XAUTH登录凭据冒充VPN网关的IKE响应;目前它没有做全职中间人的客户端部分。

资料来源：http://www.roe.ch/FakeIKEd

    -L file verbous logging to file instead of stdout
